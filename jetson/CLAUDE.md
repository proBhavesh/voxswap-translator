# VoxSwap Translator — Jetson Edition

Real-time speech-to-speech translation system running on NVIDIA Jetson Orin NX. GPU-accelerated inference enables large models and low-latency translation.

## Hardware

- **Board**: NVIDIA Jetson Orin NX (16GB RAM)
- **GPU**: 1024 CUDA cores, 32 Tensor Cores (Ampere architecture)
- **CPU**: 8-core Arm Cortex-A78AE
- **SDK**: NVIDIA JetPack 6 (Ubuntu-based, includes CUDA, cuDNN, TensorRT)
